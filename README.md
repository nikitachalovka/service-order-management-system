# Service Order System

Інформаційна система управління замовленнями та послугами, розроблена на JavaFX з використанням SQLite та Maven.

## Опис проєкту

Даний проєкт являє собою настільний додаток для управління клієнтами, послугами та замовленнями.  
Система дозволяє виконувати CRUD-операції, здійснювати пошук даних та працювати з базою даних SQLite через JDBC.

## Основні можливості

- Авторизація користувачів
- Управління клієнтами
- Управління послугами
- Управління замовленнями
- Пошук клієнтів
- Пошук замовлень
- Валідація введених даних
- Робота з реляційною базою даних

## Функціональність

### Clients

- Додавання клієнтів
- Редагування клієнтів
- Видалення клієнтів
- Пошук клієнтів

### Services

- Додавання послуг
- Редагування послуг
- Видалення послуг

### Orders

- Створення замовлень
- Видалення замовлень
- Пошук замовлень
